# UI Skills Directory

## Overview

Use this skill to pick the right UI-specialist workflow before implementing or reviewing frontend work. It condenses the ui-skills.com directory into a Codex-friendly routing guide and local catalog.

## Workflow

1. Identify the user's primary UI need: accessibility, motion, framework, testing, performance, design polish, design-system work, or broad audit.
2. Open `references/skills-catalog.md` and shortlist 1-3 matching skills by topic, trigger phrase, or provider.
3. Apply the selected skill guidance as a checklist. If multiple concerns are present, sequence them in this order:
   - framework correctness and architecture
   - accessibility and metadata
   - layout, typography, and visual system
   - motion and interaction details
   - performance and production hardening
4. Convert the chosen skill into concrete code-review criteria or implementation tasks for the current project.
5. Report which catalog entries were used and why.

## Fast Routing

- Need a general premium UI pass: start with `impeccable`, `frontend-design`, `taste-skill`, `redesign-skill`, or `ui-ux-pro-max`.
- Need accessibility: start with `audit-and-fix`, `fixing-accessibility`, `wcag-audit-patterns`, `contrast-checker`, `link-purpose`, or `use-of-color`.
- Need motion: start with `animate`, `12-principles-of-animation`, `to-spring-or-not-to-spring`, `mastering-animate-presence`, or `transitions-dev`.
- Need React/Next/Vercel: start with `react-best-practices`, `next-best-practices`, `next-cache-components`, `next-upgrade`, `shadcn`, or `react-doctor`.
- Need Vue/Nuxt: start with `vue`, `vue-best-practices`, `nuxt`, `pinia`, `vue-router-best-practices`, or `vue-testing-best-practices`.
- Need 3D/WebGL: start with the `threejs-*` entries for scene, geometry, materials, lighting, loaders, shaders, textures, animation, interaction, or postprocessing.
- Need testing/browser automation: start with `playwright-cli`, `agent-browser`, `vitest`, `vue-testing-best-practices`, or `web-quality-audit`.
- Need design-system/color/typography polish: start with `baseline-ui`, `oklch-skill`, `typeset`, `layout`, `polish`, or `swiss-design`.

## Catalog Reference

Load `references/skills-catalog.md` only when selecting a specific UI skill or when the user asks for the full skill list. The reference includes all 109 entries from the submitted ui-skills.com skills page with provider and practical Codex usage notes.
